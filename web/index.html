¿Qué es un Schema?

Un schema describe la forma y restricciones de datos. Con JSON Schema puedes especificar tipos, campos requeridos, límites, patrones y más. Esto ayuda a validar entradas, documentar APIs y generar código.

Este demo incluye un validador ligero compatible con: type, properties, required, additionalProperties, items, minItems, maxItems, enum, const, minimum, maximum, exclusiveMinimum, exclusiveMaximum, minLength, maxLength, pattern.
